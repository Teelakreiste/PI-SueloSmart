Bienvenido a SueloSmart

SueloSmart es una herramienta poderosa para el análisis de suelos. Esta aplicación proporciona funcionalidades avanzadas para calcular contenido de agua, límite líquido, límite plástico, distribución granulométrica y más.

Utilice la barra de opciones para navegar entre las diferentes secciones y realice sus análisis de suelos de manera eficiente y precisa. Si tiene alguna pregunta, consulte la ayuda proporcionada en cada sección para obtener información detallada sobre cómo utilizar las funciones de SueloSmart.

¡Gracias por elegir SueloSmart para sus análisis de suelos!
